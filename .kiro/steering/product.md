# Eventy — Product Overview

Eventy is a wedding invitation platform. Users browse templates, create and customize wedding invitations, publish them via a unique URL, and collect RSVP responses and guest wishes.

## Primary Users
- Bride / Groom / Wedding Planner — create and manage invitations

## Secondary Users
- Wedding Guests — view public invitation page, submit RSVP

## Core Flows
1. Browse invitation templates (grid with categories)
2. Select template → create invitation (bride name, groom name, date, time, venue)
3. Preview and publish invitation (generates a shareable slug URL)
4. Guests visit public link, view invitation details, submit RSVP (attending/not attending, guest count, notes/wishes)

## MVP Scope
- Anonymous invitations (no user accounts required)
- Templates are seeded data, not user-created
- Invitation statuses: draft → published
