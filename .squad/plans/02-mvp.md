# MVP Scope

Goal:

Allow a couple to create and publish a wedding invitation.

Flow:

Landing Page
→ Browse Templates
→ Select Template
→ Enter Wedding Information
→ Preview Invitation
→ Publish Invitation
→ Share Public URL

Included:

- Template Explorer
- Invitation Creation Wizard
- Invitation Preview
- Publish Invitation
- Public Invitation Page

Excluded:

- Payments
- Guestbook
- Analytics
- Subscriptions
- Custom Domains
